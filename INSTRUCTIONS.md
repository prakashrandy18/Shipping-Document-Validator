# How to Run the Shipping Document Validator

## Prerequisites
1.  **Python 3.x** installed.
2.  **Internet Connection** (for Google Gemini API).

## 1. Setup (One-time)
Open your terminal and navigate to this folder:
```bash
cd "/Users/growmax/Personal/Shipping Docs"
```

Install the required libraries:
```bash
pip3 install -r requirements.txt
```

## 2. Run the Application
Start the application server:
```bash
python3 web_app.py
```

## 3. Use the App
Open your web browser and visit:
[http://localhost:5000](http://localhost:5000)

## 4. Stop the App
To stop the server, go back to your terminal window and press **Ctrl + C**.
